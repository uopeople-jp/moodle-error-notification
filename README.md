# moodle-error-notification

### 概要
moodleのサーバーにリクエストを送りステータスコードが400番台、500番台の時にSlackに通知するアプリ

### 実行方法
GitHub Actionsで定期実行を設定

設定ファイルは.github/workflows/schedule.ymlを参照
